halo! ini web buatannya. udah dibikin rapi, clean (aesthetic), dan ga miring-miring lagi.
tetep ga pake huruf kapital ya biar lebih hidup dan natural.

cara pakenya:
1. siapin 4 foto terbaik kalian.
2. taruh di dalam folder 'photos'.
3. ganti nama fotonya jadi: 1.jpg, 2.jpg, 3.jpg, dan 4.jpg.
4. taruh 1 lagu favorit kamu di luar (sejajar sama file index.html).
5. ganti nama lagunya jadi: music.mp3.

kalau udah semua, tinggal klik 2x file index.html buat liat hasilnya di browser. 
semoga suka ya!
